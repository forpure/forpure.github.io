Hexo博客搭建

发表于 2015-12-20   |   分类于 Hexo   |  

该博客是通过Hexo搭建，部署在Github Pages，并托管在Github。之前没接触过版本管理，趁这次机会一起学习了Git。

Hexo

Hexo实际上是一个静态博客的框架。其搭建简单快速，仅需要几条命令，轻量级，并且功能比较全也比较优雅简洁，提供的主题也比较丰富。

安装配置

$ npm install hexo-cli -g
$ hexo init [blog] #不指定目录则默认当前CWD
$ [cd blog]
$ npm install
$ hexo server #本地测试，localhost:4000
  1. 安装Hexo前需要先安装Node.js和Git
  2. 在命令行输入上方命令进行安装
  3. Hexo博客配置，修改博客目录下_config.yml（全站配置）及主题目录/themes下_config.yml（主题相关配置）
  4. Hexo博客主题更换，将整个主题目录拷贝到你的Hexo博客目录下/themes中，修改全站配置_config.yml中字段：
    theme: your-theme-name
